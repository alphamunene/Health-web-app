# Vitalize Wellness App

A polished, single-page wellness web experience focused on daily health habits, personal insights, and lightweight user interaction.

## Overview
Vitalize is a frontend-first wellness app landing page built as a standalone HTML file. It blends a modern UI, clear information hierarchy, and dashboard-style sections to present wellness content in an engaging way.

## Features
- Full responsive single-page layout
- Login overlay UI and guest access flow
- Sticky top navigation and action pills
- Hero section with wellness-focused calls to action
- Stats strip cards for quick insights
- Rich visual system with custom CSS variables
- Bootstrap 5 utility support
- Bootstrap Icons integration
- Google Fonts typography pairing (DM Serif Display + Inter)

## Tech Stack
- HTML5
- CSS3 (custom properties and component styling)
- Bootstrap 5.3.3 (CDN)
- Bootstrap Icons 1.11.3 (CDN)
- Google Fonts (CDN)

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/alphamunene/Health-web-app.git
   cd Health-web-app
   ```
2. Open `index.html` directly in your browser.

No build step or package install is required.

## Deployment
Because this is a static page, it can be deployed easily to:
- GitHub Pages
- Netlify
- Vercel (static)
- Any static file host
